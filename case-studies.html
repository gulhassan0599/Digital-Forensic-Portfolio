Corporate Insider Data Theft

Scenario

Objectives: Identify how proprietary source code was leaked to a competitor.

Investigation: Analyzed the ex-employee's workstation. Timeline analysis revealed mass file copies to a Kingston USB drive on their final day. Registry artifacts confirmed the device serial number. Prefetch files showed execution of unauthorized archiving software.

Outcome: Provided concrete evidence that led to a successful legal injunction against the former employee.

Ransomware Incident Response

Scenario

Objectives: Determine patient zero, containment, and recovery path for a mid-sized healthcare provider.

Investigation: Conducted live response and memory acquisition. Identified an exposed RDP port as the initial vector. Found lateral movement via compromised admin credentials. The malware was identified as a variant of LockBit.

Outcome: Isolated affected subnets, recovered critical databases from offline backups, and patched the vulnerability within 48 hours.
