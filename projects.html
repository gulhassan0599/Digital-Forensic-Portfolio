Digital Forensics Projects

Hands-on investigations and practical laboratory work

USB Recovery Investigation

Analyzed a suspect USB drive to recover deleted financial records and determine connection timestamps.

Tools: Autopsy, FTK Imager, USBDeview

Findings: Recovered 14 deleted spreadsheets; proved connection during off-hours.

Windows Memory Analysis

Investigated a RAM dump from a compromised Windows 10 workstation to identify a hidden rootkit.

Tools: Volatility, WinDbg

Findings: Extracted malicious DLL injected into svchost.exe; recovered attacker C2 IP.

Browser Artifact Analysis

Traced user web activity by analyzing SQLite databases from Chrome and Edge to reconstruct timelines.

Tools: DB Browser for SQLite, Hindsight

Findings: Reconstructed 30-day timeline showing intent to exfiltrate data via cloud storage.

Network Traffic Analysis

Analyzed PCAP files to identify data exfiltration attempts over DNS tunneling mechanisms.

Tools: Wireshark, Zeek, NetworkMiner

Findings: Isolated command and control traffic and successfully extracted the stolen plaintext data payloads.

Mobile Device Forensics

Extracted and analyzed a physical Android image to trace deleted communications and location history.

Tools: Autopsy, ALEAPP, SQLite Browser

Findings: Recovered 200+ deleted Signal messages and mapped 3 weeks of GPS coordinates proving physical presence.

Registry Hive Investigation

Analyzed Windows Registry hives to detect advanced persistent threat (APT) persistence mechanisms.

Tools: Registry Explorer, RegRipper

Findings: Identified a hidden Run key designed to execute a malicious PowerShell payload silently on boot.
